## Common TCP Ports

- Telnet (23) -- Login
- SSH (22) -- Secure Login
- HTTP (80)
- HTTPS (443) -- Secure
- SMTP (25) -- Mail
- IMAP (143/220/993) -- Mail Retrieval
- POP (109/110) -- Mail Retrieval
- DNS (53) -- Domain Name
- FTP (21) -- File Transfer
